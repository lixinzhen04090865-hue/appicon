# appicon
🎨 纯原生 HTML5 Canvas 驱动的跨平台 App 图标生成器。0 依赖的单文件应用（SFA），本地纯前端裁剪、缩放并打包导出。拒绝臃肿框架，高三理综刷题之余的解压副产物。
# 🎨 Vanilla-Icon-Generator

> **纯原生 HTML5 Canvas 驱动的跨平台 App 图标批量生成器。**
> 拒绝臃肿，回归原生。

## 📜 缘起
同桌最近在搞一个小 App，苦于搞不定 iOS 和 Android 各种乱七八糟尺寸的图标。看着他想去下那些几十上百兆的臃肿工具，我实在受不了。正好趁着某天午休，直接用原生 Canvas 手搓了这个生成器。

同样坚持**单文件应用（SFA）**的理念，不需要安装任何环境，甚至不需要连网。下周就是苏锡常镇一模了，复习搞得焦头烂额，周末顺手把这个小工具和隔壁的 AI 客户端一起丢上来开源。

## 🛠️ 功能特性
一切都在这几百行代码里：
- **🖌️ 原生 Canvas 渲染：** 支持图片自由缩放、底色填充、透明度（Alpha）控制。
- **✂️ 智能裁剪：** 纯几何算法实现的圆形（Circle）、圆角矩形（Rounded）实时预览裁剪。
- **📦 纯前端打包导出：** 集成了轻量级打包逻辑，一键生成全平台（iOS / Android / Web / Windows）几十种尺寸的图标并打成 `.zip` 包下载。

## 🚀 使用方法
不需要克隆仓库，直接下载 `appicon.html`，双击用任何现代浏览器打开即可。本地处理，图片绝不会上传到任何服务器。
